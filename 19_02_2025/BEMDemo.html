This is a simple card about BEM architecture

BEM stands for Block Element Modifier. It is a methodology that helps you to create reusable components and code sharing in front-end development.

BEM is not a framework. It is a set of rules that you can apply to your CSS and HTML. BEM is a naming convention for classes in HTML and CSS. It stands for Block, Element, Modifier.

BEM class names are written in the format block__element--modifier. The block represents the higher-level component in a block of code. The element represents a descendant of the block that helps form the block as a whole. The modifier represents a different state or version of the block.
